# QuickCite Privacy Policy

Last updated: February 27, 2026

## Summary
QuickCite does not collect, store, or transmit any personal data.

## Data Storage
- Copy history (last 20 citations) is stored locally on your device using Chrome's storage API
- No data is sent to external servers
- No user accounts or authentication required

## External Requests
- The only external request is to the CrossRef API (https://api.crossref.org)
- This request sends only the DOI (Digital Object Identifier) of the paper
- No personal information is included in this request

## Permissions
| Permission | Purpose |
|-----------|---------|
| activeTab | Read paper metadata from the current page |
| clipboardWrite | Copy citations to clipboard |
| storage | Save copy history locally |
| scripting | Read HTML meta tags from publisher pages |

## Contact
If you have questions about this privacy policy, please open an issue on this GitHub repository.
